# Student Performance Analyzer & Improvement Recommender

## Overview
This project analyzes student academic and skill data using Machine Learning and provides personalized improvement suggestions.

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Decision Tree / Random Forest Classifier

## Features
- Analyzes student performance level (Low / Medium / High)
- Provides improvement recommendations
- Ethical and student-friendly approach

## Dataset
Includes CGPA, projects, internships, skills, and attendance.

## Outcome
Helps students identify areas of improvement rather than predicting failure.
